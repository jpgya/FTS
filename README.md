# Fortnite Training Support (FTS)

**FTS** は、フォートナイトの練習をもっと効率的・快適に行うための非公式サポートツールです。

## 主な機能

- 編集・エイム・ピースコントロールなどの練習マップを一覧表示  
- VODレビュー用のページで、動画を見ながら振り返りや分析  
- 重要なお知らせをシュッと出るテロップで通知  
- モバイルでも見やすく、スムーズなUI

## お知らせ

現在「FTS」はマップの読み込みに関する不具合を修正中です。  
**VOD機能などは問題なく利用可能です。**

## 今後の展望

- タグや検索による絞り込み  
- 練習マップのお気に入り登録機能  
- 練習進捗の手動記録機能（構想段階）
